What it does

From sliders to machine-ready files

⬡

Voronoi Tessellation

SciPy-powered diagram clipped to a right-triangle boundary with forced corner seeds for a clean, gap-free layout.

⚙️

Kerf Compensation

Cell Gap slider applies a Shapely buffer offset so adjacent edges stay the exact distance apart on the physical material.

◻️

Logo Cutout

A rounded-rectangle logo pocket is subtracted from the tessellation area — no cell overlaps the logo zone.

📐

DXF R2010 Export

Three named layers (OUTLINE, LOGO, CELLS) map directly to machine operations — cut, pocket, or engrave independently.

🔵

Corner Rounding

Shapely buffer shrink/expand rounds every vertex to the specified radius, preventing stress concentrations at sharp corners.

📉

Polyline Simplification

Douglas–Peucker tolerance collapses near-collinear vertices, keeping file size small without visible shape distortion.

Configurability

All parameters

Parameter Range Default Description
num_seeds 5 – 60 20 Number of Voronoi seed points
space_between_cells 2 – 30 mm 10 Gap between adjacent cells (kerf compensation)
outline_margin 5 – 40 mm 10 Inset from the triangle boundary
triangle_size 200 – 700 mm 476 Right-triangle leg length
corner_radius 0 – 30 mm 10 Rounding applied to cell vertices
simplification_tolerance 0.05 – 2.0 0.10 Douglas–Peucker polyline simplification

Run Locally

Full configurator — any parameters

The interactive configurator runs on Flask. Clone the repo and start it in under a minute:

# clone
git clone https://github.com/YOUR_USERNAME/voronoi-configurator.git
cd voronoi-configurator

# install dependencies
pip install -r requirements.txt

# start
python app.py

# open in browser
→ http://localhost:5000
